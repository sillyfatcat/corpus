# Corpus Pages

Static staging site for Corpus, suitable for free hosting on GitHub Pages.

## Files

- `index.html`: public landing/support page
- `privacy.html`: privacy policy page
- `styles.css`: shared styling
- `assets/app-icon.png`: app icon

## Publish On GitHub Pages

1. Create a GitHub repository for this folder.
2. Push these files to the repository root.
3. In GitHub, open `Settings` -> `Pages`.
4. Set `Source` to `Deploy from a branch`.
5. Choose your default branch and `/ (root)`.
6. Save, then wait for GitHub to publish the site.
